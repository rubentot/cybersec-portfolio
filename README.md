# Cybersecurity Portfolio

Pre-Masters preparation portfolio mapped to the NTU MSCS curriculum. This repository documents 12 weeks of hands-on cybersecurity projects covering offensive security, defensive operations, and specialist topics.

## Portfolio Structure

| Week | Project | Skills Demonstrated |
|------|---------|-------------------|
| 1 | [Home Lab Setup & Linux Hardening](week-01-lab-setup/) | VMware, Linux administration, firewall configuration, SSH hardening, system auditing |
| 2 | [Networking Fundamentals & Packet Analysis](week-02-packet-analysis/) | Wireshark, TCP/IP, DNS, ARP, HTTP traffic analysis |
| 3 | [Cryptography Hands-On](week-03-cryptography/) | AES, RSA, TLS handshake analysis, hash cracking with Hashcat |
| 4 | [Web Application Vulnerabilities](week-04-web-app-vulns/) | DVWA, OWASP Juice Shop, SQL injection, XSS, CSRF |
| 5 | [Vulnerability Scanning & Enumeration](week-05-vuln-assessment/) | Nmap, OpenVAS, vulnerability assessment reporting |
| 6 | [Penetration Testing](week-06-pentesting-methodology/) | HackTheBox CTF writeups, structured pentesting methodology |
| 7 | [SIEM & Log Analysis](week-07-SIEM-/) | Splunk, detection rules, dashboards, log ingestion |
| 8 | [Incident Response Simulation](week-08-incident-response/) | NIST SP 800-61, breach simulation, detection and containment |
| 9 | [Malware Analysis](week-09-malware-analysis/) | Static and dynamic analysis, IOC extraction, REMnux |
| 10 | [Digital Forensics](week-10-forensics/) | Autopsy, disk image analysis, artifact recovery, forensic reporting |
| 11 | [Cloud & Zero Trust Security](week-11-cloud-security/) | AWS security audit, zero trust architecture, misconfiguration remediation |
| 12 | [Portfolio Assembly & Cert Prep](week-12-portfolio/) | Documentation, Security+ preparation, MSCS curriculum mapping |

## Lab Environment

- **Hypervisor:** VMware Workstation Pro
- **Attack Machine:** Kali Linux (4GB RAM, 2 CPUs)
- **Target/Server:** Ubuntu Server 24.04 LTS (4GB RAM, 2 CPUs)
- **Host System:** Windows PC, 30GB RAM

This portfolio maps directly to 10 out of 14 NTU MSCS courses:

- SE6001 Computer Security — Weeks 1, 4
- SE6002 Application Security — Weeks 4, 5, 6
- SE6003 Cryptography — Week 3
- SE6004 Security & Risk Management — Week 12
- SE6011 Network Security — Week 2
- SE6013 Software Security — Week 6
- SE6014 Security Monitoring — Weeks 7, 8
- SE6015 Malware — Week 9
- SE6016 Forensics — Weeks 8, 10
- SE6020 Mobile, Cloud, Zero Trust — Week 11
